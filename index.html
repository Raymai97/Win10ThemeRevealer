Reveal the hidden themes in Windows 10!

It seems some people can't stand the plain white title bar in Windows 10 Build 10240 just like me. Fortunately, I have found a way to get back my colorful experience, thanks to the winaero website (http://www.winaero.com/blog/get-colored-title-bars-in-windows-10/). To make this more easier to the others, I have written a program for it so everyone can have their eye-candy in no time. :)
